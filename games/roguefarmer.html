Rogue Farmer

Year: 2024
Engine: Unity
Platform: Linux, Windows
My part: everything

Rogue Farmer is a demo state solo project by me. I made it to raise my old school grades in early 2024.
The game's idea is to farm potatoes and defend yourself and your crop from the waves of ratmen who come at night.
I did everything myself. the textures, 3D-models, programming, music, only for sound effects I used edited CC0 licensed sounds from freesound.org
The name comes from the Roguelike game genre, The game is in a very unfinished state and would have featured some roguelike elements like item upgrades.

Download

The game can be downloaded for Linux and Windows from the itch.io page.
note: the game is in really rough state
